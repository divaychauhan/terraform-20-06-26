GCP DevOps CI/CD Deployment

This project demonstrates a complete DevOps workflow using Docker, GitHub Actions, Artifact Registry, Workload Identity Federation, and Google Cloud Run.

✅ Website is successfully deployed on Google Cloud Run

Tools Used

HTML & CSS

Frontend website created using static HTML and CSS.

Docker

Website is containerized using Nginx Docker image.

GitHub Actions

CI/CD pipeline automatically builds and deploys the app.

Artifact Registry

Docker images are stored securely in Google Artifact Registry.

Cloud Run

Container is deployed as a serverless service on GCP.

IAM & WIF

GitHub authenticates securely using Workload Identity Federation.

CI/CD Pipeline Flow

1. Developer pushes code to dev branch
2. GitHub Actions workflow starts automatically
3. Docker image is built from app folder
4. Docker image is pushed to Artifact Registry
5. Cloud Run deploys the latest image
6. Website becomes live publicly
